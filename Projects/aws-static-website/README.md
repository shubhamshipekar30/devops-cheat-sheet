# AWS Static Website Hosting

## 📌 Overview

This project shows how to host a static website using AWS S3 and CloudFront.

## 🛠️ Services Used

* Amazon S3
* CloudFront

## 🚀 Steps

1. Created S3 bucket
2. Uploaded index.html
3. Enabled static website hosting
4. Added bucket policy
5. Connected CloudFront

## 🌐 Output

Live website hosted with HTTPS

## 💡 Learning

* Static hosting
* CDN (CloudFront)
* AWS permissions
